# 📓 Task Manager (Journal Project)

A backend project built with **Go**, **Gin**, and **GORM** that serves as a task and journal manager.  
This project was developed as a **pet project** to showcase my backend development skills and to prepare for a junior Golang developer role.  

---

## ✨ Features
- 🔑 **User Authentication**
  - Secure login and registration with **gRPC** and **JWT**  
  - Passwords hashed with **bcrypt**
- 📖 **Task & Journal Management**
  - RESTful **CRUD endpoints** to create, read, update, and delete journal entries
- ⚡ **High Performance**
  - Built on **HTTP/2** and gRPC for fast, reliable communication
- 🗄️ **Database**
  - Persistent storage with **SQLite**  
  - Managed via **GORM ORM**
- 🛠️ **Testing & Debugging**
  - API endpoints tested and documented with **Postman**

---

## 🚀 Tech Stack
- **Go (Golang)**  
- **Gin Framework**  
- **GORM ORM**  
- **SQLite Database**  
- **gRPC & Protocol Buffers**  
- **JWT Authentication**  
- **Postman for API testing**  
- **HTTP/2**
